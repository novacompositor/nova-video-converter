# Nova Compositor — Task Checklist (Edit Room / Phase 2)

Данный документ описывает план работ по превращению комнаты Edit в полноценный, но легковесный и быстрый NLE-редактор, вдохновленный базовым воркфлоу Premiere Pro.

## Цель:
Создать быстрый, интуитивно понятный редактор для сборки черновика (рыбы), нарезки клипов и подготовки таймлайна к дальнейшему композитингу, цветокоррекции или VFX.

## 1. Подготовка UI (Очистка и Контекстные меню)
- [x] Удалить объединяющую синюю кнопку "Import XML" из `ProjectPanel`/`Media Bin`.
- [x] Добавить контекстное меню для **пустой области** Media Bin (по правому клику):
  - [x] `Import Media...` (вызывает FileDialog)
  - [x] `Import XML...` (вызывает загрузку FCPXML)
  - [x] `New Sequence...` (создание таймлайна)

## 2. Создание и управление Секвенциями (Таймлайнами)
- [x] Реализовать UI-диалог "New Sequence" (Настройки: Имя, Разрешение, Frame Rate, Sample Rate).
- [x] Интегрировать команду `EngineCommand::CreateSequence` в Rust-бэкенд.
- [x] Обновить UI `EditRoom`, чтобы при выборе Секвенции в Media Bin она открывалась на таймлайне и в Program Monitor (Viewer).

## 3. Базовый монтаж на Таймлайне (Drag & Drop, Нарезка, Перемещение)
- [x] Поддержка Drag & Drop из Media Bin на область Timeline (создание нового клипа).
- [x] Отрисовка базовых треков (Video 1, Video 2, Audio 1).
- [x] Отображение блоков клипов на таймлайне с именами файлов.
- [x] Перетаскивание клипов влево/вправо по времени (Selection Tool - `V`).
- [x] Подрезка (Trimming) краев клипа — визуальные ручки trim-in / trim-out на каждом клипе.
- [x] Разрезание клипа (Razor Tool - `C`) → `SplitClipInSequence` в бэкенде.
- [x] Удаление клипов (`Delete`/`Backspace`) → `RemoveClipFromSequence` в бэкенде.

## 4. Воспроизведение и Program Monitor
- [x] Синхронизация Playhead на таймлайне с отображаемым кадром во `ViewerPanel`.
- [x] Кнопки переключения режимов вьювера (Source Monitor / Program Monitor).
- [x] Пробел (Play/Pause) плавно проигрывает смонтированную секвенцию.

## 5. Продвинутые функции сборки
- [ ] Выделение нескольких клипов (мульти-селект).
- [ ] Nesting (Precompose) — объединение выделенных клипов в новую Секвенцию.
- [ ] Магнитная привязка (Snapping) клипов к другим клипам и к Playhead.

## Технический стек для Edit Room:
- **UI:** QML ListView / Repeater для треков и клипов. Custom C++ модели, если QML-модели станут медленными.
- **Engine (Rust):** Доработка `Sequence`, `Track`, `Clip` структур в `project_schema`. Эффективный расчет "какой медиафайл показывать на текущем кадре таймлайна".
- **Media:** Чтение конкретных кадров из файлов с помощью `media_ffmpeg` на основе `in_point` и `out_point` клипа.
